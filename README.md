## 🚀 Featured Project: TOPS (Temporal Ops)

> *Infrastructure intelligence platform for SREs and Platform Engineers*

**TOPS** is a production-grade platform solving causal reasoning across infra and GenAI systems to reduce MTTR. It exposes an **MCP server** (`tops-mcp`) as its primary interface — usable directly in Claude Desktop, Cursor, or Windsurf.

**Stack:** FastAPI · Next.js · LangChain (RAG) · LangGraph (stateful agents) · Anthropic SDK · AWS Bedrock · pgvector · Upstash Kafka · Supabase

**Agent architecture:**
- `Minutemen` — High-frequency collectors (Claude Haiku)
- `TemPad` — Causal reasoning engine (Claude Sonnet)  
- `Pruning Blade` — Counterfactual simulation (Claude Opus)
